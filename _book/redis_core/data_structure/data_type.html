数据类型

redis提供如下数据类型：

  1. String（字符串）
  2. List（列表）
  3. Hash（哈希）
  4. Set（集合）
  5. Sorted Set（有序集合）

简单来说，redis底层数据结构一共6种：

  1. 简单动态字符串
  2. 双向链表
  3. 压缩列表
  4. 哈希表
  5. 跳表
  6. 整数数组
